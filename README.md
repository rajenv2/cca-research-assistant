# CCA Research Assistant

Multi-agent research assistant with a custom MCP backend, shipped through CI.
Built as the hands-on project for the Claude Certified Architect — Foundations (CCA-F) exam.

## Components
- MCP server (tools + resources) — Domain 4
- Orchestrator + subagents — Domain 1
- GitHub Actions CI running Claude Code — Domain 2
- Error handling + human-in-the-loop escalation — Domain 5

## Setup
\`\`\`bash
python3.12 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
\`\`\`
